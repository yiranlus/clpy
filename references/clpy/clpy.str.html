Function (new-from v)¶

Create a Unicode object from a another object..

V should be already encoded in Unicode. If you want to create a Unicode object from a normal string, use :cl:function:new.

LEFT string should be an Unicode object, RIGHT can be a string or an Unicode object.

Function (tail-match unicode substr &key (start 0) end (direction :forward))¶

Return T if SUBSTR matches unicode[start:end] at the given tail end.

DIRECTION can be :FORWARD which mean a suffix match; or :BACKWARD which means prefix match.

Function (rich-compare left right op)¶

Rich compare two Unicode strings.

The return value can be T, NIL, or Py_NotImplemented. Possible values for OP are :GT, :GE, :EQ, :NE, :LT, :LE.
