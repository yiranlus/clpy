CLPy Documentation¶

CLPy is a Common Lisp wrapper of Python Limited API. It use cl-autowrap to generate the FFI bindings and then rewrap these functions to a more Lisper way. CLPy also comes with a CLthon, which is a set of macros that facilitate writing Python-like scripts in Common Lisp.
